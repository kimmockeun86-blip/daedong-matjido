# 제품 요구사항 정의서 (01_PRD)

이 파일은 대동맛지도 프로젝트의 기능 스펙 및 무자본 구동 요구사항을 기입하는 요구사항 사양서입니다.

---

## 1. 개요
- **서비스명**: 대동맛지도 (Neon Cyberpunk Gourmet Map)
- **목표**: 사용자가 맛집 엑셀 데이터를 업로드하면 지도상에 네온 사이버펑크 스타일 마커로 표시하고, 각종 미식 툴킷(코스 계획, 월드컵, 룰렛 등)을 제공하며, 미식 일기장을 통해 맛집 방문 기록을 관리할 수 있는 무자본 기반 모바일/웹 서비스.

## 2. 핵심 원칙
- **영구 무자본 (Zero-Budget)**: 서버 유지비용이 없는 Client-side 아키텍처. LocalStorage, HTML5 Web APIs, OpenStreetMap 등 무료 기술 스택만 사용.
- **실전 배포 수준의 품질**: 데모가 아닌 마켓 출시 및 배포가 바로 가능한 완성도 높은 빌드.

## 3. 기능 명세
- **엑셀 파서 & 로컬 캐시 지오코딩**: 엑셀 업로드 및 주소-좌표 변환(Nominatim API 준수, 로컬스토리지 캐싱).
- **네온 하이라이트 Leaflet 지도**: Leaflet 기반 지도 렌더링 및 마커 setIcon() 하이라이팅.
- **미식 툴킷**: 룰렛 추천, 맛집 월드컵, 코스 플래너, Dutch Pay 계산기, 공유용 딥링크.
- **락/해금 시스템**: 미식 일기장 작성 및 공유 횟수에 따라 Top 10 시크릿 노포 맛집 해금.
